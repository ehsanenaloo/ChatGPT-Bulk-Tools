# ChatGPT Bulk Tools Clean+

这是一个仅本地运行的 Chrome 扩展，用于批量选择、归档和删除 ChatGPT 对话。

## 特点
- 为可见对话添加复选框
- 支持可见选择、反选、清空、按标题筛选、最旧可见选择
- 支持 Dry Run 预览
- 支持滚动时自动附加复选框
- 保留了已经验证可用的快速删除/归档核心逻辑
- 在弹窗中显示操作日志

## 隐私
- 无追踪
- 无支付逻辑
- 无远程请求
- 无 `chrome.identity`
